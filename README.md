# Invitacion gamer romantica

Proyecto estatico listo para publicarse y compartirse por link.

## Estructura

- `index.html`
- `styles.css`
- `app.js`

## Publicar en Netlify (mas rapido)

1. Entra a [https://app.netlify.com/drop](https://app.netlify.com/drop)
2. Arrastra esta carpeta completa (`invitacion_gamer_romantica`)
3. Espera a que termine el deploy
4. Copia el link publico (`https://...netlify.app`) y compartelo

## Publicar en GitHub Pages

1. Crea un repositorio nuevo en GitHub
2. Sube `index.html`, `styles.css` y `app.js`
3. Ve a `Settings > Pages`
4. En `Build and deployment`, elige:
   - Source: `Deploy from a branch`
   - Branch: `main` y carpeta `/ (root)`
5. Guarda y espera el deploy
6. Tu link quedara como:
   - `https://TU-USUARIO.github.io/TU-REPO/`

## Prueba local (opcional)

Solo abre `index.html` con doble clic en tu PC.
